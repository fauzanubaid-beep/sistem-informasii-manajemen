Pertemuan 4

Sistem Manajemen Database

A. Konsep Dasar Sistem

Pada pertemuan ini akan dijelaskan mengenai definisi database, media dan sistem penyimpanan data, sistem pengolahan data, dan organisasi database.

Tujuan Pembelajaran

  • Definisi database
  • Media dan sistem penyimpanan data
  • Sistem pengolahan data
  • Organisasi database

B. Uraian Materi

1. Definisi Database

Database mulai berkembang sejak tahun 1960-an. Charles William Bachman merupakan salah satu pelopor dalam bidang database dan mendapatkan Turing Award atas kontribusinya.

Database dapat didefinisikan sebagai kumpulan data yang saling berhubungan dan disimpan secara terintegrasi untuk mendukung kebutuhan informasi organisasi.

Kelebihan Database

  • Meningkatkan akses dan respon data
  • Meningkatkan keamanan data
  • Data dapat digunakan bersama
  • Meningkatkan independensi data

Kekurangan Database

  • Biaya implementasi tinggi
  • Pengelolaan relatif kompleks
  • Biaya konversi dan pelatihan

2. Media dan Sistem Penyimpanan Data

Media penyimpanan data meliputi pita magnetik, disk magnetik, dan media optik seperti CD dan DVD. Pengelolaan database dilakukan oleh DBMS.

3. Sistem Pengolahan Data

Sistem pengolahan data melibatkan proses batch processing dan online processing yang mendukung operasional organisasi.

4. Organisasi Database

Organisasi database bertujuan untuk memastikan data tersimpan secara terstruktur, aman, dan mudah diakses. Model organisasi data meliputi hierarchical, network, dan relational model.
